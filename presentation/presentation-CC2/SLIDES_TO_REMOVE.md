# SLIDES À SUPPRIMER - Hors sujet CC2.2

## Slides techniques trop détaillées pour une présentation stratégique :

1. **Architecture Base de Données** (ligne 332-366)
   - Raison : Détails techniques non pertinents pour la direction
   
2. **Structure du Projet** (ligne 367-458)
   - Raison : Organisation du code source = hors sujet stratégique

3. **5 Slides Interface utilisateur** (lignes 420-487)
   - Raison : Démonstration trop longue, 1 slide suffit
   
4. **Pipeline ETL** (ligne 489-536)
   - Raison : Détails techniques déjà couverts dans "Synthèse des Données"

5. **FinOps & Infrastructure** (à simplifier ou retirer)
   - Si conservé : intégrer brièvement dans les recommandations

6. **RGPD/Souveraineté** (à simplifier)
   - Si conservé : 1 slide max sur la conformité

## RÉSULTAT : Au lieu de ~20+ slides, viser 12-15 slides max

Conformément aux consignes d'une présentation de 20 minutes.
